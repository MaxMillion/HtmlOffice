¿Que es ODF Reader?

Es una aplicación para Firefox OS que te permite leer cualquier archivo en formato OpenDocumentFormat(ODF) almacenado en la memoria SD de tu dispositivo. OpenDocument es un formato de archivo abierto y estándar para el almacenamiento de documentos ofimáticos tales como hojas de cálculo, textos, gráficas y presentaciones

Que tipo de formatos puedo leer?

Los Documentos soportados son los que poseen las extensiones .odt para documentos de texto, .ods para hojas de cálculo, .odp para presentaciones

Como puedo convertir documentos de Microsoft Office en OpenDocument?

La aplicacion de Microsoft Office, posee un sistema de conversion y guardado para convertir cualquier documento ya sea word, power point o excel en OpenDocument, pronto en futuras actualizaciones añadiremos un sistema para convertir documentos directo desde ODF Reader

Los documentos no se ven como en Microsoft Office?

Existe algunas caracteristicas de Microsoft Office, no compatibles con OpenDocument por lo tanto, algunas cosas no se visualizaran de manera correcta, para ver cuales son las compatibilidades visita: Microsoft.com
